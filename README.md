# 💼 Portfolio

> Ezekiel Laguiab's updated personal portfolio website — built with HTML, CSS, and JavaScript.

---

## 📖 About

**Portfolio** Ezekiel Laguiab's personal portfolio website. It showcases projects, skills, and background as a developer, with a refined design and improved user experience over the original version. Built entirely with vanilla web technologies for a fast, dependency-free experience.

## ⚡ Getting Started

No build tools or dependencies needed — just clone and open!

```bash
# Clone the repository
git clone https://github.com/EzekielLaguiab/Portfolio2.0.git

# Navigate into the project folder
cd Portfolio2.0

# Open in your browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.

---

## 🌐 Deploying with GitHub Pages

1. Go to your repository **Settings**
2. Navigate to **Pages** under the *Code and automation* section
3. Under **Branch**, select `main` and click **Save**
4. Your portfolio will be live at `https://ezekiellaguiab.github.io/Portfolio2.0/`
